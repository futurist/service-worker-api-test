# service-worker-api-test
service-worker-api-test for sw API availability

## Install

```sh
npm i service-worker-api-test
```

## Usage

```js
const serviceWorkerApiTest = require('service-worker-api-test')
// return a Promise
serviceWorkerApiTest()
  .then(results=>{
    console.log(results)
  })
  .catch(err=>{
    alert(err)
  })
```
